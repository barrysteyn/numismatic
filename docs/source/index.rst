Numismatic
==========
Numismatic is tool for querying crypto coin exchanges on the command line. Numismatic
has the following design goals:

 * **Extensible** - it is easy to add new feeds (exchanges).
 * **Websocket first** - whereever possible, websockets are used above rest calls.
 * **Standardizes output** -
 * **Simple cli**

Quick Start
-----------

To get general command line help::

    coin
